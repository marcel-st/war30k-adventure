# TPS Vision - Death Guard 30K

This migration evolves the original SDL2 top-down prototype into a third-person shooter focused on 30K-era Death Guard combat fantasy.

## Core Pillars

- Heavy, deliberate Astartes movement and weapon feel.
- Over-shoulder third-person shooting with objective-driven encounters.
- Warhammer 30K visual identity: worn ceramite, gothic military ruins, smoke-heavy atmospherics.
- Story continuity from Garro's warning run while shifting to 3D mission spaces.

## Vertical Slice Definition

- One playable 3D level: Isstvan extraction drill.
- Functional systems: movement, camera, aim, boltgun firing, reload, HUD, objective completion.
- Placeholder geometry and simple materials tuned toward Death Guard palette.

## Initial Death Guard Direction

- Off-white and muted green armor blocks with metallic wear.
- High roughness and grime bias in materials.
- Cold ambient lighting with selective toxic-green accents.
- Broad armored silhouette with oversized pauldrons.
