# WAR30K Adventure v0.6.0 — Legacy SDL2 Baseline

`v0.6.0` remains the final release from the SDL2-focused development track before the
larger Godot TPS migration work that now lives on `main`.

## Release focus

- Retro top-down rendering pipeline improvements for the SDL2 game.
- Mission flow and spawn reliability fixes.
- Documentation refresh for the C++/SDL2 architecture.

## Notes for current `main`

Since this release tag, `main` has moved far beyond the SDL2 baseline and now includes:

- Godot 4 TPS vertical-slice gameplay (`tps/`)
- Multi-wave mission director with boss encounter
- Story chapters, contact dialogues, and cutscene support
- Controller-ready input mapping for gameplay and narrative flow
- Progression/ability/settings systems
- Integrated open-license audio pipeline and runtime audio manager

See `README.md` and `CHANGELOG.md` for current implementation details.
